# Tic Tac Toe AI

This project is a Tic Tac Toe AI game developed using Python as part of the CodSoft Artificial Intelligence Internship.

---

##  Features

- Player vs Computer gameplay
- AI-based computer moves
- Score tracking system
- Replay functionality
- Interactive console interface

---

## Technologies Used

- Python
- Random Module
- Time Module

---

## How to Run

Open terminal and run:

```bash
python tictactoe.py
```
